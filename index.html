Tempo integration notes · GitHub-only workflow
📄 Read‑only helper
This is a single index.html file with inline CSS and JavaScript — perfect for hosting on GitHub Pages with no local tooling or build steps.
Deploy from GitHub (web only)
  1. In the GitHub web UI, create a new public repository (for example, tempo-stablecoin-dex).
  2. Add a new file named index.html in the root, and paste this entire file into it, then commit directly to the main branch.
  3. Open the repository’s Settings → Pages, choose the main branch and the / (root) folder, then save. GitHub will build your static site automatically.
  4. After a short delay, your DEX UI will be available at the GitHub Pages URL shown in the Pages settings — everything done purely in the browser.
To turn this mock into a real Tempo DEX client, replace the simulation logic in the script with calls to Tempo’s TypeScript SDKs (Viem/Wagmi extensions) and the stablecoin exchange APIs. ([docs.tempo.xyz](https://docs.tempo.xyz/sdk/typescript) [docs.tempo.xyz](https://docs.tempo.xyz/guide/stablecoin-dex) [docs.tempo.xyz](https://docs.tempo.xyz/guide/stablecoin-exchange/executing-swaps))
Tempo testnet quick facts
  • Network: Tempo Testnet (Moderato). Chain ID: 42431. RPC: https://rpc.moderato.tempo.xyz. Explorer: https://explore.tempo.xyz. ([docs.tempo.xyz](https://docs.tempo.xyz/quickstart/connection-details))
  • Default test stablecoins: pathUSD, AlphaUSD, BetaUSD, ThetaUSD — faucet‑minted TIP‑20 tokens for testing. ([docs.tempo.xyz](https://docs.tempo.xyz/quickstart/faucet) [docs.tempo.xyz](https://docs.tempo.xyz/learn/tempo/native-stablecoins))
  • Tempo’s enshrined DEX lets you trade between USD stablecoins with deterministic routing via quote tokens (often pathUSD), optimizing pricing for payments. ([docs.tempo.xyz](https://docs.tempo.xyz/protocol/tip20/overview) [docs.tempo.xyz](https://docs.tempo.xyz/protocol/exchange/pathUSD) [docs.tempo.xyz](https://docs.tempo.xyz/protocol/exchange/quote-tokens) [docs.tempo.xyz](https://docs.tempo.xyz/protocol/exchange))
  • TIP‑20 extends ERC‑20, so existing EVM tools (Viem, Wagmi) work out of the box, with extras like transfer memos and policy‑based compliance hooks. ([docs.tempo.xyz](https://docs.tempo.xyz/protocol/tip20/overview))
  • Use the official TypeScript SDKs and Tempo transaction extensions to query pools, get DEX quotes, submit swaps, and pay gas directly in your preferred stablecoin. ([docs.tempo.xyz](https://docs.tempo.xyz/sdk/typescript) [docs.tempo.xyz](https://docs.tempo.xyz/learn/tempo/native-stablecoins))
